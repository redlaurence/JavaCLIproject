# JavaCLIproject ☕

A simple Command Line Interface (CLI) project built with Java. This project demonstrates the basics of Java programming,
user interaction through the terminal, and object-oriented programming concepts.

---

## 📌 About the Project

This project is a beginner-friendly Java application that simulates a coffee ordering system through the command line.

It was created as a learning project to practice:

- Java syntax and structure
- Classes and objects
- User input handling
- Conditional statements and loops
- CLI-based application flow

---

## 🚀 Features

- Interactive command-line menu
- Coffee selection system
- User-friendly terminal interaction
- Simple and clean Java implementation

---

## 🛠️ Technologies Used

- Java
- NetBeans IDE
- Apache Ant
